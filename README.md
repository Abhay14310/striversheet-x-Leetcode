# DSA Practice: Striver Sheet × LeetCode

A comprehensive collection of Data Structures and Algorithms (DSA) solutions and practice problems based on Striver's SDE Sheet and LeetCode problems.

## 📚 Overview

This repository contains JavaScript implementations of fundamental DSA concepts, patterns, and solutions to help prepare for coding interviews and competitive programming.

## 📁 Repository Structure

```
.
├── README.md                      # Project documentation
└── StriverSheet/                  # Main practice directory
    ├── Basics/                    # Fundamental concepts and control flow
    ├── Patterns/                  # Pattern generation and printing
    ├── Strings/                   # String manipulation
    └── Core Concepts/             # Pass by reference, scope, etc.
```

- **Patterns**: Various pattern printing and visualization problems

## 🚀 Getting Started

### Prerequisites
- Node.js installed on your system
- Basic JavaScript knowledge

### Running Files

To run any JavaScript file, use the following command:

```bash
node StriverSheet/filename.js
```

**Example:**
```bash
node StriverSheet/pattern2.js
node StriverSheet/string.js
```
